PHASE 7 — PHYSICS ENGINE (MOST CRITICAL SYSTEM)
════════════════════════════════════════════════════════════

Implement a FULLY CUSTOM physics engine in physics.js. Do NOT use
Box2D, Matter.js, or any third-party library. Build from scratch.

CORE PHYSICS REQUIREMENTS:

  Fixed Timestep: 1/60 second (16.667ms) with accumulator-based loop
  Substepping: 3–8 substeps per frame for stability under fast impacts
  Integration: Semi-implicit Euler (velocity Verlet for birds)
  Interpolation: Render uses interpolated positions between physics steps

COLLISION DETECTION:
  Broad Phase: Spatial hash grid (cell size = 2x largest object radius)
  Narrow Phase: SAT (Separating Axis Theorem) for convex polygons
  Circle-Circle: Exact analytical detection
  Circle-Polygon: GJK algorithm
  CCD (Continuous Collision Detection): Swept tests for fast projectiles
  Sleeping Bodies: Bodies with velocity < threshold sleep after 0.5s still
  Wake Events: Sleeping bodies wake when impacted or structurally connected
    body moves

RIGID BODY PROPERTIES (per body):
  position (vec2), velocity (vec2), acceleration (vec2)
  angle, angularVelocity, angularAcceleration
  mass, inverseMass, inertia, inverseInertia
  restitution (bounciness), staticFriction, dynamicFriction
  isStatic (immovable ground/walls), isSleeping, isSensor
  shape: circle | polygon | compound

CONSTRAINTS:
  Distance constraints (for slingshot elastic band)
  Point constraints (pivot joints for hinged structures)
  Position correction: Baumgarte stabilization (beta = 0.2)
  Velocity correction: 8–12 iteration constraint solver per substep

IMPULSE RESOLUTION:
  Resolve overlaps with position projection
  Apply normal + tangential impulses
  Angular impulse from offset contact points
  Restitution handled per-material pair

GRAVITY: 980 units/s² (scaled to feel like 9.8 m/s²)
AIR RESISTANCE: Small quadratic drag on all airborne bodies
GROUND FRICTION: Per-material static/dynamic values

MATERIAL PHYSICS PROPERTIES:
  ┌─────────────┬──────┬──────┬────────────┬──────────┬────────────┐
  │ Material    │ HP   │ Mass │ Restitution│ Friction │ Density    │
  ├─────────────┼──────┼──────┼────────────┼──────────┼────────────┤
  │ Wood        │ 100  │ 1.0  │ 0.20       │ 0.60     │ medium     │
  │ Bamboo      │ 70   │ 0.6  │ 0.35       │ 0.50     │ light      │
  │ Glass       │ 40   │ 0.5  │ 0.10       │ 0.30     │ light      │
  │ Brick       │ 200  │ 2.0  │ 0.15       │ 0.70     │ heavy      │
  │ Concrete    │ 350  │ 3.0  │ 0.05       │ 0.80     │ very heavy │
  │ Steel       │ 500  │ 4.0  │ 0.25       │ 0.40     │ very heavy │
  │ Energy Shield│ 150 │ 0.1  │ 0.95       │ 0.00     │ near-zero  │
  │ Bamboo+Mud  │ 120  │ 1.2  │ 0.10       │ 0.75     │ heavy-mid  │
  └─────────────┴──────┴──────┴────────────┴──────────┴────────────┘

DAMAGE SYSTEM:
  Damage = impulse magnitude × bird damage multiplier × material_vulnerability
  Structural integrity: HP reduced by damage; visual cracks appear at 50% and 25% HP
  Material-specific damage multipliers per bird type
  Chain-collapse: if a block's support is removed, gravity topples the stack
  Scoring: (initial HP - remaining HP) × material_score_factor

SLINGSHOT PHYSICS:
  Band modeled as spring: F = -k(extension) × direction, k = 850
  Max pull distance: 120 units from pivot
  Release velocity = extension × spring_constant / bird_mass
  Angle capped to -80° to +20° elevation range
  Rubber band renders as catenary curve between two fork tips + bird
  Drag trajectory preview: dotted arc with 40 points, Verlet integration

PHASE 16 — PERFORMANCE REQUIREMENTS
════════════════════════════════════════════════════════════

TARGET: Stable 60 FPS on mid-range devices (2020+ Android/iOS)

OPTIMIZATION RULES:
  Object pooling: Reuse particle, pig, bird, and block objects
  Off-screen culling: Skip draw/update calls for objects outside viewport
  Dirty rectangles: Only redraw changed areas when possible
  Batch rendering: Group same-material draws, minimize state changes
  Physics sleeping: Non-moving bodies skip physics computation
  Particle budget: Enforce max 2000 active particles; oldest die first
  Audio pooling: Reuse AudioBufferSource nodes; max 16 simultaneous SFX
  Image caching: All sprite canvas drawings cached to offscreen canvases
  Layer caching: Background parallax layers cached between frames
  GC avoidance: Pre-allocate vec2/mat2 objects, avoid new in hot loops
  Profiling hooks: Built-in FPS counter + draw call counter (toggleable)
  RequestAnimationFrame: All rendering tied to RAF, never setInterval

LOADING:
  All assets load with a Bengali progress screen with progress bar
  Non-blocking asset loading (async/await pattern)
  Core game starts with minimal assets; sprites load progressively

PHASE 17 — SPECIAL GAME MECHANICS
════════════════════════════════════════════════════════════

TRAJECTORY PREDICTION:
  Use the exact same physics integration as gameplay (Verlet)
  Preview arc regenerates every 2 frames during drag
  Shows collision point if trajectory hits a structure (red dot marker)

SLOW MOTION:
  Triggered automatically: special bird activate = 0.3× game speed
  Time scale affects: physics dt, particle speeds, animation timing
  Audio: pitch shifts down during slow-mo (Web Audio playbackRate)
  Visual: slight vignette + motion blur streaks on fast objects

DESTRUCTION SYSTEM:
  Blocks fracture in stages: Crack 1 (50% HP), Crack 2 (25% HP), Destroy
  Each material has unique fracture polygon split (wood = splinter lines,
  glass = crystalline cracks, brick = rectangular chunks)
  Debris: when block destroyed, spawn 3–8 small physics debris pieces
  Debris affects other blocks physically (can chain destroy pigs below)
  Debris despawn: fade out after 3 seconds if on ground (performance)

BOUNCE / RICOCHET:
  Glass has high restitution — birds bounce off glass at steep angles
  Steel also deflects birds partially
  Energy shields: birds bounce back at near-full speed
  Fun outcomes where bounced birds kill other pigs (bonus score)

COMBO SYSTEM:
  Pig pop within 2 seconds of another = COMBO + score multiplier
  1-pig: normal, 2-pig: "COMBO! ×1.5", 3+: "MULTI-KILL! ×2.0"
  Max combo: 5+ pigs in 2 seconds = "FLOCK ANNIHILATION! ×3.0"

BONUS BIRDS:
  Unused birds at end of level = +3000 score per bird
  Animated: unused birds peck ground and explode in coin confetti
